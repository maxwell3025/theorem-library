We define and prove properties of basic algebraic structures.

\begin{theorem}
The sum of two even numbers is even.
\end{theorem}

\begin{proof}
Let $a$ and $b$ be even numbers. Then $a = 2k$ and $b = 2m$ for some integers $k$ and $m$.
Therefore, $a + b = 2k + 2m = 2(k + m)$, which is even.
\end{proof}
